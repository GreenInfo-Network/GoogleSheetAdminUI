1. Go to the Export JSON menu and select "Export as JSON".
  2. In the text area with the JSON code, right-click and Select All, then right-click and Copy.
  3. Open the admin UI in another browser window.
  4. In the admin UI, select all text and empty it out. Then paste in this new JSON code you copied earlier.
  5. In the admin UI, click Save. The changes will be visible on the map one minute later.
